# 🏠 Eduardo Calabuig - Plataforma Web

## 📌 Descripción del Proyecto

Este proyecto consiste en el desarrollo de una plataforma web completa para un diseñador de interiores, **Eduardo Calabuig**, cuyo objetivo es centralizar su presencia digital y ofrecer una experiencia profesional tanto para clientes como para la gestión interna.

La aplicación combina una web pública orientada a usuarios con un panel administrativo (backend) que permite la gestión de contenido de forma dinámica.

---

## 🌐 Funcionalidades Principales

### 🖼️ Portfolio

* Visualización de proyectos de interiorismo
* Presentación visual atractiva de trabajos realizados
* Organización por categorías o estilos

### 📰 Blog / Noticias

* Publicación de artículos y novedades
* Gestión de contenido desde el panel administrativo
* Actualización constante del sitio web

### 🛍️ Ecommerce

* Catálogo de productos (estatuas, esculturas, decoración, etc.)
* Visualización detallada de cada producto
* Sistema preparado para expansión comercial

### 🔐 Panel Administrativo (WebApp)

* Acceso privado para el administrador
* Gestión completa de:

  * Productos
  * Proyectos
  * Noticias
* Interfaz pensada para facilitar la creación y edición de contenido

---

## 🏗️ Arquitectura del Proyecto

El sistema está dividido en dos grandes partes:

### Frontend

* Interfaz pública accesible para los usuarios
* Diseño orientado a la experiencia visual

### Backend / Admin Panel

* Aplicación web privada
* Gestión de datos y contenidos

## 🚀 Instalación y Uso

```bash
# Clonar el repositorio
git clone https://github.com/JavierCornejoLeal/eduardoCalabuig.git

# Entrar al proyecto
cd eduardoCalabuig

# Instalar dependencias
npm install

# Ejecutar en desarrollo
npm run dev
```
